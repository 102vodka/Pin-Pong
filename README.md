      🏓 Проект: Ping-Pong на Pygame 
Простая реализация классической аркады «Пинг-Понг» для двух игроков, написанная на языке Python с использованием библиотеки Pygame.
      📋 Функционал
Управление двумя игроками: Левая ракетка (W/S) и правая ракетка (Стрелки).
Динамика: Мяч ускоряется на 10% при каждом ударе о ракетку.
Система счета: Игра ведется до 10 очков.
Экран завершения: Возможность перезапустить игру клавишей R или выйти через ESC.
      🛠 Технологии
Python 3.x
Pygame (библиотека для создания игр)
      🎮 Управление
Игрок	Вверх	Вниз
Игрок 1 (Слева)	W	S
Игрок 2 (Справа)	↑	↓
R — Начать заново (после окончания матча).
ESC — Выход из игры.
      🚀 Как запустить
Установите Python (если еще не установлен).
Установите библиотеку Pygame через терминал:
bash
pip install pygame
Используйте код с осторожностью.
Сохраните код в файл (например, main.py) и запустите его:
bash
python main.py
Используйте код с осторожностью.
      🏗 Структура кода
GameSprite: Базовый класс для создания игровых объектов (мяч, ракетки) с методами отрисовки.
Player: Класс, наследующий GameSprite, содержащий логику перемещения ракеток.
Основной цикл: Обрабатывает столкновения мяча, границы экрана и систему начисления очков
