# 🏓 Игра "Ping-Pong"

Простая и динамичная реализация классической игры Пинг-Понг для двух игроков на языке **Python**.

## 📖 Описание проекта
Этот проект демонстрирует использование библиотеки **Pygame** для создания игровой логики: обработки столкновений, управления объектами и системы подсчета очков.

### Особенности игры:
- **Ускорение мяча:** При каждом ударе о ракетку мяч ускоряется на 10%.
- **Графика:** Спрайты генерируются программно (без внешних картинок).
- **Система победы:** Игра продолжается до тех пор, пока один из игроков не наберет **10 очков**.

---

## 🎮 Управление


| Действие | Игрок 1 (Слева) | Игрок 2 (Справа) |
| :--- | :--- | :--- |
| **Движение вверх** | `W` | `Клавиша Вверх` |
| **Движение вниз** | `S` | `Клавиша Вниз` |

**Общие команды:**
* `ESC` — выйти из игры.
* `R` — начать заново (после победы одного из игроков).

## 💻 Логика кода
Код организован с помощью объектно-ориентированного программирования (ООП):
1. **`GameSprite`** — базовый класс, отвечает за создание «тел» объектов (мяча и ракеток) и их отрисовку.
2. **`Player`** — подкласс для ракеток, содержит алгоритмы перемещения.
3. **Игровой цикл** — проверяет столкновения (`sprite.collide_rect`), отскоки от границ и условия гола.

## 📦 Установка и запуск
1. Убедитесь, что у вас установлен Python.
2. Установите библиотеку Pygame:
   ```bash
   pip install pygame
   ```
3. Запустите игру:
   ```bash
   python main.py
   ```
